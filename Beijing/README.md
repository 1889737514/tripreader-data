# 北京市 Beijing

## 数据 Data

### 轨道交通（含BRT）

识别方式：1E文件，线路和站点

数据文件：metro-tu.csv

### 公交

识别方式：1E文件，线路和站点前4位，16进制

## 备注 Remark

北京一卡通（非交联卡）暂无法识别轨道站点和公交线路。
